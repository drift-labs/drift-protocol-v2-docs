# Data API Playground

At a high level, the Drift Data API is the backbone for reading historical and real-time protocol data so that apps, analytics, and integrations don’t need to index the chain themselves. The Drift Data API follows the OpenAPI spec.

<Callout type="info">
All endpoints are relative to `https://data.api.drift.trade`.
</Callout>
